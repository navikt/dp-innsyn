# dp-innsyn

Gir deg svar på alt du lurer på

## Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

```
./gradlew build
```

## For å kjøre applikasjonen lokalt:

```
docker-compose up -d
./gradlew run
```
